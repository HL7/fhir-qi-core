RelatedArtifact

Related artifacts such as additional documentation, justification, or bibliographic references.

= Must Support, = Is Modifier, = QiCore defined extension

FieldCard.TypeDescription
type1..1StringThe type of relationship to the related artifact.
display0..1StringA brief description of the document or knowledge resource being referenced, suitable for display to a consumer.
citation0..1StringA bibliographic citation for the related artifact. This text SHOULD be formatted according to an accepted citation format.
url0..1StringA url for the artifact that can be followed to access the actual content.
document0..1AttachmentThe document being referenced, represented as an attachment. This is exclusive with the resource element.
resource0..1ResourceThe related resource, such as a library, value set, profile, or other knowledge resource.
